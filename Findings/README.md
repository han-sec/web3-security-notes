# Security Findings

A curated collection of real-world security vulnerabilities across different Web3 protocol types.

## Organization

Findings are organized by protocol type to help identify patterns and common issues within each category.

### [Cross-Chain](./cross-chain/)
Vulnerabilities in cross-chain bridges, relayers, and interoperability protocols.

**Examples:**
- Message spoofing in cross-chain gateways
- Relayer exploits in bridge protocols
- Cross-chain action manipulation

### [Infrastructure](./infrastructure/)
Core blockchain and network-level vulnerabilities including consensus, P2P networking, and node operations.

**Examples:**
- Consensus protocol bugs
- Network DoS attacks
- Authentication failures in validator networks
- Cryptographic implementation issues

### [DeFi](./defi/)
Decentralized finance protocol vulnerabilities (coming soon).

**Examples:**
- AMM exploits
- Lending protocol vulnerabilities
- Yield farming issues
- Flash loan attacks

## How to Use

1. Browse by protocol type to understand category-specific attack patterns
2. Each finding includes technical details, root cause analysis, and mitigation strategies
3. Use these as reference material when auditing similar protocols

## Contributing

When adding new findings:
1. Place them in the appropriate protocol category
2. Include clear technical details and root cause analysis
3. Link to official disclosures or postmortems when available
4. Follow the existing documentation format
